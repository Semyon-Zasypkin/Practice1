1) для создания локального репозитория нужен гит. Устанавливаем его

Для проверки вводим в командную строку git --version
Покажет версию гита

2)Нужно показать гиту за каким проектом следить(отслеживал изменения файлов)

Переходим в папку с проектом cd и путь

Если наше приложение находится на другом диске придётся использовать /d

cd /d "путь"

3) git init (инициализация изменений) 

.gitignore (в ней можно написать файлы, которые не будут отслеживаться гитом)

git add . (добавление всех файлов в пулл отслеживаемых)

git status (статус)

git commit -m "ИМЯ ТОЧКИ" (создание точки восстановления) Нужно сделать обязательно !

4) связывание git remote add origin "ссылка"


5)git push -u origin main (отправка на удалённые репозиторий)
