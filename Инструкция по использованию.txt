Инструкция по использованию приложения:

1)Загрузка изображения
Нажмите кнопку «Загрузить изображение».
В диалоговом окне выберите файл изображения (.png, .jpg).

2)Снимок с камеры:
Нажмите «Сделать снимок с камеры».
Если веб-камера подключена, приложение сделает снимок и отобразит его.
При ошибке (например, камера не доступна) появится сообщение об ошибке.

3)Выбор цветового канала:
В приложении есть кнопки отвечающие за вырезания каналов, нажимаю можно переключатся между каналами

4)Изменение размера:
Нажмите «Изменить размер».
Введите ширину и высоту в пикселях в появившихся диалоговых окнах.
Изображение изменит размер, если введены корректные положительные числа.

5)Понижение яркости:
Нажмите «Понизить яркость».
Введите значение
Изображение станет темнее.

6)Рисование круга:
Нажмите «Нарисовать круг».
Введите координаты центра круга, (x,y) радиуса или оставьте пустыми для использования центра изображения(есть параметры по умолчанию).
Круг рисуется красным.

7)Выход:
«Выход» для закрытия приложения
